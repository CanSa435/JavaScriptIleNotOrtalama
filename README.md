📊 Not Ortalama Sistemi

Bu proje, JavaScript kullanılarak geliştirilmiş basit bir not ortalama hesaplama uygulamasıdır.

Özellikler:
- Vize ve final notlarını kullanıcıdan alma
- Ortalama notu hesaplama
- Başarılı / Başarısız durumunu gösterme
- Form doğrulama işlemleri

Bu proje JavaScript, HTML ve DOM manipülasyonu pratiği yapmak amacıyla geliştirilmiştir.
